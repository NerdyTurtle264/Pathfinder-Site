Character Randomizer

So, you want to make a character but don’t have a starting place. Here is a list of tables to help you find out who you are, where you are from, and how you act.

Name and Basic identity

Here are some random ways to make a name. To keep it fair, you may only remove one letter or flip a single pair of them.
  • A normal name but alphabetized example: Peter- Eeprt, Eliza-Aeilz, Madison- Aidmnos
  • Reverse a standard name Mike-Ekim, Mary-Yram, John-Nhoj
  • Take a word in a different language, like Latin, and convert it into a name with a meaning. Be sure to be respectful of the culture the name is taken from. Don’t use the names of deities or cultural figures that may hurt another person. An example of a good one: Tenebris=darkness and Calidum=warm- Calida Tenebrae= Warm Darkness
  • If you want a truly randomized name, follow the steps below.

For feminine sounding names, start and/or end with a vowel. In English, feminine names contain more soft sounds.

For masculine names start and/or end the name with a consonant. Masculine names tend to be harsher sounding and more clipped.

  1. Roll a d20 for the consonants and a d6 for the vowels.
  2. Repeat this pattern as desired. Feel free to switch or replace something if you don’t like it.
  3. Shorter is better. Aim for a first name 3-6 letters long and a last name of 4-8 letters long. Your fellow players will thank you if they can pronounce it.
